# Frequently Asked Questions

## Softnix Docs ใช้แทน GitHub wiki ได้ไหม

ใช้ได้ถ้าต้องการ documentation portal ที่มี versioning, search, AI-readable exports และ MCP integration มากกว่า wiki ทั่วไป

## ทำไมต้องเก็บเอกสารใน Git

เพราะ Git ช่วยให้มี history, branch, review, Pull Request และ audit trail ที่ชัดเจน เหมาะกับเอกสาร product ที่เปลี่ยนตาม software lifecycle

## เขียนภาษาไทยได้ไหม

ได้ เนื้อหา MDX รองรับ UTF-8 จึงเขียนภาษาไทยและอังกฤษร่วมกันได้

## AI agents แก้เอกสารได้แค่ไหน

AI agents สามารถอ่าน ค้นหา และสร้าง draft changes ได้ แต่ไม่สามารถ publish เอกสารโดยตรง

## Semantic search มีแล้วหรือยัง

v1 ใช้ full-text search ก่อน และเตรียม schema สำหรับ embedding ไว้สำหรับ phase ถัดไป

## เอกสารนี้เป็นตัวอย่างการจัดการ docs จริงไหม

ใช่ เอกสารชุดนี้ถูกเก็บใน `docs-repo` ตาม canonical layout และถูก portal อ่านเหมือน product documentation อื่น
