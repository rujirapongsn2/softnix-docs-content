# Softnix Logger Overview

Softnix Logger collects application, infrastructure, and security logs so support teams can find operational issues quickly.

## Capabilities

- Collect logs from Softnix products
- Configure Syslog forwarding
- Retain searchable troubleshooting history
- Export evidence for support cases

```bash
softnix-logger status
```
